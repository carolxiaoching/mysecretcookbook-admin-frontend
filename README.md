# My Secret CookBook

- 此為 "我的秘密食譜" 前端後台
- 僅供練習，無任何商業用途

## 使用套件

- vite v5.3
- vue v3.4
- pinia v2.1
- mitt v3.0
- gh-pages v6.2
- axios v1.7
- vue-axios v3.5
- bootstrap v^5.3
- bootstrap-icons v1.11
- vee-validate v4.13
- @vee-validate/i18n v4.13
- @vee-validate/rules v4.13
- apexcharts v3.54
- vue3-apexcharts v1.7
- vue-loading-overlay v6.0
- vuedraggable v4.1

## 專案指令列表

- 安裝套件
  ```
  npm install
  ```
- 執行開發模式
  ```
  npm run dev
  ```
- 編譯
  ```
  npm run build
  ```
